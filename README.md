# scoop-bucket

Personal Scoop bucket for packages published by AspireOne.

## Usage

Add the bucket once:

```powershell
scoop bucket add AspireOne https://github.com/AspireOne/scoop-bucket
```

Then install any app from this bucket:

```powershell
scoop install <app>
```

## Apps

- `codex-manage`

For example:

```powershell
scoop install codex-manage
```
